#Mini Hospital Triage System (Python Project)

A simple rule-based Python system that simulates how hospitals perform initial patient screening (triage) before doctor consultation.

This project was built as part of my Python learning journey focusing on conditional logic and real-world decision systems.

##Project Overview

This system collects patient information and uses logical rules to:
- Check eligibility for assessment
- Calculate BMI (Body Mass Index)
- Classify health condition
- Validate patient ID using modulo operation
- Assign basic triage priority level

##Features

- Patient data input (name, age, weight, height, ID)
- BMI calculation using Python arithmetic
- ID validation using modulo operator (%)
- Eligibility check using logical operators (and)
- Health classification using if / elif / else
- Nested decision structure for triage priority

##Concepts Used

- if / elif / else statements  
- Nested conditions  
- Logical operators (and)  
- Modulo operator (%)  
- Basic arithmetic operations  

##How It Works

1. User enters patient details  
2. System calculates BMI  
3. System checks if patient ID is valid  
4. System checks eligibility (age + ID)  
5. System classifies health condition  
6. System assigns triage priority level  

##Example Use Case
This system simulates how hospitals or clinics perform initial patient screening before assigning doctors or treatment priority.
